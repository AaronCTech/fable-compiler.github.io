JsFunc

Namespace: Fable.Core
Parent Module: JsInterop

Use it to cast dynamic functions coming from JS. If you know the argument and return types, use System.Func<> instead. If you need a constructor (must be applied with new keyword), use JsConstructor.

Sample

1: 
2: 

let f: JsFunc = import "myFunction" "./myLib"
f.Invoke(5, "bar")
val f : obj
